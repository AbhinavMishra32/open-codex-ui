# Surface Release Notes

Each release version has a directory named after the exact package version.

Example:

- `releases/0.1.0-alpha.1/web.md`
- `releases/0.1.0-alpha.1/electron.md`
- `releases/0.1.0-alpha.1/tui.md`
- `releases/0.1.0-alpha.1/api.md`
- `releases/0.1.0-alpha.1/agent-core.md`

`npm run release:check` validates these files exist for the current version.
